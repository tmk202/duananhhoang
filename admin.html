🌐 GitHub — host free mãi mãi + thêm sản phẩm online

GitHub Pages chỉ là file tĩnh, không tự ghi. Admin dùng GitHub API để commit products.json + ảnh vào repo → Pages tự deploy. Miễn phí, không hết hạn (trong gói free GitHub).

Tạo tại GitHub → Settings → Developer settings → Personal access tokens.
Cần quyền Contents: Read and write (fine-grained) hoặc scope repo (classic).
Token chỉ lưu trên trình duyệt này (localStorage), không ghi vào repo.
  1. Push code lên GitHub, bật Settings → Pages (branch main / root)
  2. Tạo token → dán vào form trên → Lưu & bật GitHub
  3. Vào Thêm sản phẩm → upload ảnh → Lưu
  4. Đợi 1–2 phút, F5 trang shop — SP mới hiện
